/src/constants/api.js:
Export API_BASE pointing to the
Flask API at /api/v1

/src/services/api.js:
One function per API endpoint
covering all /api/v1/ routes
that were just created.
Store JWT in AsyncStorage.
Add token to all protected requests.
Handle 401 by clearing token
and navigating to login.

/src/context/AuthContext.js:
Wraps the app with auth state.
On startup check AsyncStorage for token.
Validate token with GET /api/v1/auth/me.
Provide: user, token, login,
logout, register, isLoading.

/src/context/LocationContext.js:
GPS location state.
Provide: coords, locationName,
requestLocation.
Use expo-location.

  "How was your session?"
  Rate Now button → ReviewScreen

  Update conversation unread count
  Show notification banner if
  not on MessageScreen

  Show notification banner
  Update booking status in list
