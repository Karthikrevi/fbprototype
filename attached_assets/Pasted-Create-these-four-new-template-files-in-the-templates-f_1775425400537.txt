FILE 1: templates/journal_entry.html
Route: /erp/finance/journal-entry
Same purple gradient styling as erp_dashboard.html

Form POSTing to /erp/finance/journal-entry:
- entry_date date picker default today
- entry_type radio: debit or credit
- account dropdown:
  Cash, Bank, Sales Revenue,
  Cost of Goods Sold, Inventory,
  Accounts Receivable, Accounts Payable,
  Operating Expenses, Platform Fees,
  Capital, Drawings, Fixed Assets,
  Tax Payable, GST Collected, GST Paid
- amount number minimum 0.01
- description text required
- sub_category text optional
- reference_number text optional
- Save Entry submit button

Table below showing {{ recent_entries }}:
Date, Type, Account, Amount, Description, Source
Green Manual badge where entry_source = manual
Grey Auto badge for all others

Back button to /erp/reports

FILE 3: templates/invoice_create.html
Route: /erp/finance/invoice/create
Same purple gradient styling

Form POSTing to /erp/finance/invoice/create:
- customer_email text
- customer_name text
- invoice_date date picker
- due_date date picker
- payment_terms dropdown:
  Due on Receipt, Net 7, Net 15, Net 30

Line items table with JavaScript:
Columns: Description, Qty, Unit Price, Total
One row by default
Add Item button adds new row without reload
Each row has Remove button
JavaScript auto-calculates:
  Row total = qty x unit price
  Subtotal = sum of all row totals
  GST = subtotal x gst_rate / 100
  Grand total = subtotal + GST

- gst_rate number field default 18
- notes textarea optional
- Create Invoice submit button

All calculations in JavaScript only
No page reload for adding or removing items

Back button to /erp/reports
