Section 1 — Pet Owner Details:
  owner_name text required
  owner_email text optional
  owner_phone text required
  owner_address textarea
  owner_city text
  emergency_contact text

Section 2 — Pet Details:
  pet_name text required
  species dropdown:
    Dog, Cat, Bird, Rabbit,
    Fish, Hamster, Reptile, Other
  breed text
  gender dropdown: Male, Female,
    Unknown, Neutered Male,
    Spayed Female
  date_of_birth date picker
  weight number with kg label
  color text
  distinguishing_marks textarea
  microchip_id text
  allergies textarea
  medical_conditions textarea
  insurance_info text

  already exists in pet_owners

  Insert new pet with owner_id
  Flash: "Patient [name] registered"
  Redirect to patient_detail

PART 2 — SOAP Medical Record routes:

Add route
/furrvet/patients/<int:pet_id>/medical/add
GET and POST:

GET: render furrvet/add_medical_record.html
This is the most important form
in the entire FurrVet system.
SOAP note format:

Header:
  Patient name and species (read only)
  Vet name from session (read only)
  Visit date date picker default today
  Appointment link dropdown (optional)
    shows today's appointments for
    this patient

SUBJECTIVE section:
  Chief Complaint textarea required
    placeholder: What is the main
    reason for today's visit?
    e.g. Not eating for 2 days,
    limping on left front leg
  Owner observations textarea
    placeholder: What has the owner
    noticed at home?
  Duration of symptoms text
    placeholder: How long has this
    been happening?
  Medical history notes textarea
    placeholder: Any relevant past
    conditions or medications?

OBJECTIVE section:
  Vitals subsection:
    Temperature number field
      with °C label
      normal range shown: 37.5-39.2°C
      Highlight red if outside range
    Weight number with kg label
    Heart Rate number with bpm label
      normal range: 60-140 bpm dogs
    Respiratory Rate number
      with breaths/min label
    BCS score dropdown 1-9:
      Body Condition Score
      1=Emaciated, 5=Ideal, 9=Obese
    Mucous Membranes dropdown:
      Pink/Moist (Normal),
      Pale, White, Yellow/Jaundiced,
      Blue/Cyanotic, Brick Red, Tacky
    Capillary Refill Time dropdown:
      <2 seconds (Normal),
      2-3 seconds, >3 seconds

  Physical Examination textarea
    placeholder: Document findings
    from head to tail examination.
    Include: eyes, ears, nose, mouth,
    lymph nodes, thorax, abdomen,
    musculoskeletal, skin, neurological

ASSESSMENT section:
  Diagnosis text input required
    placeholder: Primary diagnosis
  Differential Diagnoses textarea
    placeholder: Other possible
    diagnoses to consider
  Prognosis dropdown:
    Excellent, Good, Fair,
    Guarded, Poor, Grave

PLAN section:
  Treatment Plan textarea required
    placeholder: What treatments
    are being administered today?
    Include medications, procedures,
    supportive care
  Prescription textarea
    placeholder: Medications to send
    home with owner. Include drug name,
    dose, frequency, duration
    e.g. Amoxicillin 250mg 1 tablet
    twice daily for 7 days
  Follow Up Instructions textarea
    placeholder: What should the owner
    watch for at home? When to return?
  Follow Up Required checkbox:
    If checked show follow_up_date
    date picker
  Dietary Recommendations textarea
    optional
  Activity Restrictions textarea
    optional

POST:
  Insert into medical_records table
  If follow_up_required:
    Insert into appointments with
    status = scheduled
    appointment_type = Follow Up
  If prescription not empty:
    Call bridge_furrvet_to_reminder()
    to create pet reminder in
    main erp.db for pet parent
  Flash: "Medical record saved"
  Redirect to patient_detail

PART 3 — Vaccination routes:

Add route
/furrvet/patients/<int:pet_id>/
vaccinations/add
GET and POST:

GET: render furrvet/add_vaccination.html
Form fields:
  vaccine_name dropdown:
    Dog vaccines:
      Rabies, DHPP (Distemper/Hepatitis/
      Parvovirus/Parainfluenza),
      Bordetella, Leptospirosis,
      Lyme Disease, CIV (Canine Influenza)
    Cat vaccines:
      FVRCP (Feline Viral Rhinotracheitis/
      Calicivirus/Panleukopenia),
      Rabies, FeLV (Feline Leukemia),
      FIV, FIP, Chlamydia
    Show relevant vaccines based on
    pet species if species known
    Always show Other option
  If Other: vaccine_name text input
  vaccine_type dropdown:
    Core, Non-Core, Booster
  batch_number text required
  manufacturer text
  vaccination_date date default today
  next_due_date date
    auto-calculate based on vaccine:
    Rabies: +1 year or +3 years
    DHPP: +1 year after initial
    Others: +1 year default
  site_of_injection dropdown:
    Left Neck, Right Neck,
    Left Hind Leg, Right Hind Leg,
    Left Shoulder, Right Shoulder
  adverse_reactions textarea optional

POST:
  Insert into vaccinations table
  Mark as verified in pawsport_documents
  in main erp.db if pet owner email
  matches a FurrButler account:
    INSERT INTO pawsport_documents
    document_type = Vaccination Records
    verified = 1
    verified_by = furrvet
    issued_by = clinic_name from session
  Call bridge_furrvet_to_reminder()
    for next due date reminder
  Generate vaccination certificate
    (PDF or printable HTML)
  Flash: "Vaccination recorded"
  Redirect to patient_detail

Add route
/furrvet/patients/<int:pet_id>/
vaccinations/<int:vacc_id>/certificate
GET:
  Generate printable vaccination
  certificate with:

    Clinic stamp placeholder
  Render as printable HTML page
  Add print button
