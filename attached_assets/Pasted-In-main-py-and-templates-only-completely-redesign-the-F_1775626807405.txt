POST handler for all upload types:
  Determine upload type from form
  Save file to static/uploads/furrwings/
  Based on type:
    Vaccine: insert into pet_reminders
      with source = furrwings_vet
      insert into pawsport_documents
      with verified = 1
      verified_by = furrwings_vet
    Prescription: insert into
      pet_reminders for each drug
      source = furrwings_vet
    Microchip: update pet in Replit db
      lock the field
    Lab results: insert into
      pet_reminders as reference
    Health cert: insert into
      health_certificates table
      update pawsport_documents
    General: insert into
      pawsport_documents
  Send SocketIO notification to
  pet parent:
    event: new_vet_document
    message: "Dr [name] has sent
    [document type] for [pet name]"
  Log to furrwings_vet_activity
  Return JSON:
    success: true/false
    message: confirmation text
    redirect: next URL
