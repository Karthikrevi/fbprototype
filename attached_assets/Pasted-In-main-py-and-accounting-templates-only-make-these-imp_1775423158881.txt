PART 4 — Fix Balance Sheet hardcoded values:

In the balance_sheet route replace all
hardcoded values with real database queries:

Fixed assets: Pull from a new table
CREATE TABLE IF NOT EXISTS fixed_assets (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    vendor_id INTEGER,
    asset_name TEXT,
    purchase_value REAL,
    current_value REAL,
    purchase_date TEXT,
    asset_type TEXT,
    FOREIGN KEY (vendor_id) REFERENCES vendors(id)
)

If no fixed assets recorded show 0.

Owner equity: Pull from a new table
CREATE TABLE IF NOT EXISTS capital_accounts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    vendor_id INTEGER,
    entry_type TEXT,
    amount REAL,
    description TEXT,
    entry_date TEXT DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (vendor_id) REFERENCES vendors(id)
)

If no capital recorded show 0.

Long term debt: Pull from ledger_entries
where account = 'Accounts Payable'
and sub_category = 'Long Term'.

Remove all hardcoded fallback numbers.
Show actual calculated values even if zero.
