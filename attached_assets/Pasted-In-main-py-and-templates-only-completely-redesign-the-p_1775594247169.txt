PART 1 — Database additions:

Add to init_erp_db() in main.py:

CREATE TABLE IF NOT EXISTS 
pet_travel_history (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    pet_index INTEGER NOT NULL,
    user_email TEXT NOT NULL,
    travel_type TEXT NOT NULL
        CHECK(travel_type IN 
        ('domestic','international')),
    destination TEXT NOT NULL,
    departure_date TEXT,
    return_date TEXT,
    handler_name TEXT,
    handler_id INTEGER,
    purpose TEXT,
    airline TEXT,
    flight_number TEXT,
    notes TEXT,
    created_at TEXT DEFAULT CURRENT_TIMESTAMP
)

CREATE TABLE IF NOT EXISTS
pawsport_documents (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    pet_index INTEGER NOT NULL,
    user_email TEXT NOT NULL,
    document_type TEXT NOT NULL,
    document_name TEXT NOT NULL,
    file_url TEXT,
    issue_date TEXT,
    expiry_date TEXT,
    issued_by TEXT,
    verified BOOLEAN DEFAULT 0,
    verification_hash TEXT,
    notes TEXT,
    created_at TEXT DEFAULT CURRENT_TIMESTAMP
)

PART 2 — Update pet passport route:

In main.py find the pet passport route
/pet/<int:pet_index>/passport

Update it to pull:
1. Pet details from Replit db
2. All pawsport_documents for this pet
3. Vaccination records from
   pet_reminders where
   reminder_type = Vaccination
   and source = furrvet
4. All bookings for this pet
   from bookings table
5. Handler bookings from
   handler_bookings table
6. Domestic travel history from
   pet_travel_history where
   travel_type = domestic
7. International travel history
   where travel_type = international
8. Completion percentage:
   Count verified documents /
   total required documents * 100
   Required: Microchip, Vaccination,
   Health Certificate, DGFT,
   AQCS, Quarantine Clearance

Pass all data to pawsport.html

Add route
/pet/<int:pet_index>/passport/
add-travel POST:
Save new travel record to
pet_travel_history table
Redirect back to passport

Add route
/pet/<int:pet_index>/passport/
upload-document POST:
Accept file upload
Save to static/uploads/pawsport/
Save record to pawsport_documents
Redirect back to passport
