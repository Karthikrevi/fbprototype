Create a new React Native + Expo project
that works as both a mobile app and
a web app using React Native Web.

Do not modify any Python files.
Do not modify any existing templates.
Do not delete any accounts or data.
Do not modify the Replit key-value db.

SERVICES:

/src/services/api.js:
All API calls in one place.
Functions for every endpoint:
- auth: login, register, getMe
- pets: getAll, getOne, create,
        update, getPassport
- groomers: search
- vendors: getProfile, getGroomers,
           getSlots
- bookings: getAll, create, review
- marketplace: search, getVendor
- orders: getAll, create
- handlers: getAll, getOne,
            book, getMyBookings
- chat: getConversations,
        getMessages, send, start
- community: getPosts
- stray: getAll, getOne
- location: vets, boarding, set
- groomer: getProfile

Store JWT token in AsyncStorage.
Add token to all protected requests
in Authorization header.
Handle 401 by clearing token
and redirecting to login.

/src/context/AuthContext.js:
React context for authentication state.
Provides: user, token, login, logout,
          register, isLoading
On app start check AsyncStorage for
existing token and validate with /me
If valid restore session automatically

/src/context/LocationContext.js:
React context for user location.
Provides: location, locationName,
          requestLocation, setManual
Uses expo-location for GPS permission

GDPR:

/src/screens/GDPRConsentScreen.js:
Shown on first launch only.
Stored in AsyncStorage gdpr_accepted key.
If already accepted skip this screen.

Full screen white design:
FurrButler logo at top

Section 1 — What we collect:
Your name and email for your account
Your pet information you provide
Your location when you search nearby
Your booking and order history
Chat messages with vendors

Section 2 — How we use it:
To connect you with pet services
To show relevant local vendors
To process your bookings and orders
We never sell your data to third parties

Section 3 — Your rights:
You can download your data anytime
You can delete your account anytime
You can opt out of marketing anytime
You can withdraw consent anytime

Required consent — must accept:
☐ I agree to the Terms of Service
☐ I agree to the Privacy Policy

Optional consent:
☐ I agree to receive promotional
  messages from vendors I book with
☐ I agree to share anonymous usage
  data to improve the app

Privacy Policy link opens a modal
with full privacy policy text.
Terms of Service link opens a modal
with full terms text.

Accept & Continue button:
Only active when both required
checkboxes are checked.
Saves consent with timestamp to
AsyncStorage and to server.

NAVIGATION:

/src/navigation/AppNavigator.js:
Use React Navigation.

Stack structure:
If gdpr_accepted not in AsyncStorage:
  Show GDPRConsentScreen first

If not authenticated:
  AuthStack:
    WelcomeScreen
    LoginScreen
    RegisterScreen

If authenticated:
  MainStack with bottom tab navigator:
    Tab 1: Home (HomeScreen)
    Tab 2: Discover (DiscoverScreen)
    Tab 3: Bookings (BookingsScreen)
    Tab 4: Shop (MarketplaceScreen)
    Tab 5: Profile (ProfileScreen)
