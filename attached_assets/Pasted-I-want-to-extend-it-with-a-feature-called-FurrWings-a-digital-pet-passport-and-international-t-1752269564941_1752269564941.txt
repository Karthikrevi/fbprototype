## ✅ 1. STATIC DIGITAL SIGNATURE SYSTEM (F-DSC)

### Goal:
Allow vets to upload health/vaccine docs and apply a **tamper-evident, non-IT-Act-compliant digital signature**.

### Implementation:
- Extend `passport_documents` table to include:
  - `is_signed` (BOOLEAN)
  - `doc_hash` (TEXT)
  - `signature_timestamp` (DATETIME)
- On upload:
  - If “Sign Document” checkbox is ticked, hash the document using:
    ```python
    hash = sha256(file_bytes + vet_id + timestamp).hexdigest()
    ```
  - Store the hash and set `is_signed = True`
  - Log signature info visibly in the UI

### Disclaimer (displayed on all signed docs and vet portal):
````

This digital signature is valid within the FurrButler ecosystem. It is not certified under the Indian IT Act, 2000. For export validation, documents are submitted via certified authorities or partner handlers. This system ensures document traceability, tamper protection, and vet identity verification within the FurrWings network.
````
