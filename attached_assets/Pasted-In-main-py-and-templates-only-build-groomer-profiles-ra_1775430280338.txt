PART 2 — FurrButler Certified Badge:

Add certification check function
in main.py:

def check_groomer_certification(employee_id):
    conn = sqlite3.connect('erp.db')
    c = conn.cursor()

    Check these criteria:
    1. Average overall_rating >= 4.8
       from employee_reviews
    2. Total reviews count >= 50
    3. Would book again percentage >= 85%
       (count where would_book_again = 1
       divided by total reviews)
    4. No unresolved citizen reports
       linked to this employee's bookings
    5. Employee join_date at least
       180 days ago

    Returns True if all criteria met
    Returns False otherwise

    conn.close()

Add certified_groomers table:
CREATE TABLE IF NOT EXISTS
certified_groomers (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    employee_id INTEGER UNIQUE NOT NULL,
    certified_date TEXT,
    is_active BOOLEAN DEFAULT 1,
    suspended_date TEXT,
    suspension_reason TEXT,
    FOREIGN KEY (employee_id)
        REFERENCES employees(id)
)

Add route /erp/certifications/check
POST: Called automatically when a new
review is submitted
Runs check_groomer_certification()
for the reviewed employee
If certified and not already in

If previously certified but criteria

PART 3 — Groomer of the Month:

Add groomer_of_month table:
CREATE TABLE IF NOT EXISTS
groomer_of_month (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    employee_id INTEGER NOT NULL,
    vendor_id INTEGER NOT NULL,
    month TEXT NOT NULL,
    total_reviews INTEGER,
    avg_rating REAL,
    would_book_again_pct REAL,
    FOREIGN KEY (employee_id)
        REFERENCES employees(id)
)

Add route /erp/hr/groomer-of-month/calculate
POST: Calculate Groomer of the Month
for current month per vendor

For each vendor:
Find the employee with highest combination
of reviews received this month AND
highest average rating this month
weighted score = (reviews_this_month * 0.4)
              + (avg_rating * 0.6 * 10)

Insert or update groomer_of_month
for current month

This route should be called:

- Automatically on 1st of each month

  logs in
